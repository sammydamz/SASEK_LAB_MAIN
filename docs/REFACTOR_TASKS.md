## Animation Attributes Reference

### Common Animation Types:
- `data-animate="fade-in"` - Fade in on scroll
- `data-animate="slide-up"` - Slide up from bottom
- `data-animate="slide-left"` - Slide in from left
- `data-animate="slide-right"` - Slide in from right
- `data-animate="scale-in"` - Scale up from small
- `data-animate="count-up"` - Number counting animation
- `data-animate="draw-line"` - SVG line drawing effect
- `data-animate="pulse"` - Pulsing/breathing effect

### Implementation Notes:
- Add `data-animate-delay="100"` for staggered animations (100ms, 200ms, etc.)
- Add `data-animate-duration="500"` for custom animation duration
- Add `data-animate-trigger="scroll"` or `data-animate-trigger="hover"`
